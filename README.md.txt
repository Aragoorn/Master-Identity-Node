# MasterIdentityNode Factory
این یک قرارداد هوشمند در شبکه Base است که از الگوی Factory برای دپلوی و مدیریت نودهای هویت دیجیتال استفاده می‌کند.

## ویژگی‌ها
- دپلوی هوشمند قراردادها (Proxy Factory Pattern)
- بهینه‌شده برای شبکه Base
- معماری استاندارد و مقیاس‌پذیر

## نحوه استفاده
برای دپلوی یک نود جدید، تابع `deployMasterNode` را در این قرارداد فراخوانی کنید. آدرس قراردادِ تولید شده در متغیر `deployedContract` ذخیره می‌شود.
